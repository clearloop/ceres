# Quickstart

## 0. Install `cargo-contract`

```
cargo install cargo-contract
```


## 1. Install `ceres`

```
cargo install ceres
```


## 2. Run `cargo contract new`

```
cargo contract new flipper
```


### 3. Compile with debug info

```
cargo contract build
```

### 4. Test flipper

```
ceres target/ink/flipper.contract info
```
